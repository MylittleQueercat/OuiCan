# 📰 OuiCan: Le Français par la curiosité

**OuiCan** is a high-end, editorial-style French learning platform available at **[ouican.vercel.app](https://www.google.com/search?q=https://ouican.vercel.app)**. It transforms the daily news into a sophisticated linguistic experience, combining vintage 20th-century newspaper aesthetics with modern AI-driven tools.

## ✨ Key Features

  * **Editorial Interface**: A bespoke "Newspaper Lab" design featuring Gothic typography and a wide-format layout.
  * **AI-Powered Content**: Dynamic lesson generation tailored to user interests.
  * **Visual Progress Tracking**: A refined "Editorial Ribbon" navigation bar to track learning streaks (Flamme) and proficiency levels.
  * **Interactive Design**: Precision-engineered UI with adjustable parameters for a pixel-perfect vintage feel.

## 🛠️ Technical Stack

  * **Frontend**: React.js with Styled-components/Injected CSS.
  * **Backend & Auth**: Supabase for secure user authentication and profile management.
  * **Typography**: Hosted via Google Fonts (Playfair Display, Inter, and UnifrakturMaguntia).
  * **Animation**: Custom CSS keyframes for delightful micro-interactions.

## 🐈 The "Cat Editor" Lore

Every article in OuiCan is "written" by our resident feline editor. If you see the pen dancing wildly or notice a slight delay, *c'est la faute du chat\!* (It's the cat's fault).

## 🚀 Getting Started

1.  **Visit the Site**: [ouican.vercel.app](https://www.google.com/search?q=https://ouican.vercel.app)
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Launch the Lab**:
    ```bash
    npm start
    ```
